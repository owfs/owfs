
OWNET/Pascal uses a portion of the `Ararat Synapse` TCP/IP library, necessary 
for communicating with OWServer.

The Initial Developer of the Original `Ararat Synapse` Code is Lukas Gebauer. The complete 
`Ararat Synapse` project (sources,documentation,usage lisence etc.) can be found at 
http://synapse.ararat.cz/
